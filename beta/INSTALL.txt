스마트스토어 답글 도우미 — 베타 설치 방법
========================================

1) ZIP 받기
   GitHub → Code → Download ZIP
   또는 beta/package-beta.ps1 로 만든 zip 사용

2) 압축 풀기
   폴더 이름 예: naver-smartstore-review-reply-master

3) config.js 준비
   beta/config.js 를 프로젝트 루트( manifest.json 있는 곳 )에
   config.js 이름으로 복사

4) 크롬 확장 설치
   - 주소창: chrome://extensions
   - 「개발자 모드」 켜기
   - 「압축해제된 확장 프로그램을 로드합니다」
   - 위 폴더 선택

5) 사용 시작
   - 퍼즐 아이콘 → 「스마트스토어 답글 도우미」 고정
   - 확장 열기 → [설정] → [가입하기] → 이메일·비밀번호(8자+)
   - [구독하기] → 「테스트 결제」 (베타)
   - [리뷰] / [상품문의] 탭에서 판매자센터 데이터 가져와 답글 만들기

※ 「개발자 확장 프로그램」 경고는 베타라서 나옵니다. [확장 프로그램 관리]에서 허용해 주세요.

문의: GitHub Issues 또는 배포자에게 연락
