# 스마트스토어 리뷰 답글 도우미

네이버 스마트스토어 판매자센터 리뷰 답글용 Chrome 확장 프로그램 (Manifest V3).

## 설치

1. `config.example.js`를 `config.js`로 복사하고 Gemini API 키 입력
2. Chrome → `chrome://extensions` → 개발자 모드 → **압축해제된 확장 프로그램 로드**
3. 이 폴더 선택

## 사용 흐름

1. 팝업에서 리뷰 엑셀 업로드
2. 생성할 리뷰 선택 → **답변 생성**
3. **답글 검토하기**에서 수정 → **일괄 확인 및 적용 활성화**
4. 판매자센터 리뷰 답글창에서 자동 입력
